# Van Runner - 2D Platform Oyunu

Sifirdan JavaScript ve HTML5 Canvas ile gelistirilmis,
tamamen tarayicida calisan 2D platform oyunu.

## Oyun Hakkinda

Van Runner, engellerden kacarak hayatta kalmaya calisan
bir platform oyunudur. Seviye arttikca engeller hizlanir
ve oyun zorlasir.

## Ozellikler

- Saf JavaScript ile yazildi (framework yok!)
- HTML5 Canvas ile gercek zamanli render
- Cift zipplama ozelligi
- 3 farkli engel turu (normal, ucuyor, platform)
- Partikul efektleri
- Yildizli arka plan animasyonu
- Skor, en yuksek skor, seviye ve can sistemi
- Seviye arttikca hizlanan zorluk
- Oyuncu animasyonu

## Nasil Oynanir

- Sol/Sag ok: Hareket
- Yukari ok veya Bosluk: Zipla
- Cift yukari ok: Cift Zipla (havada ikinci sans!)
- Enter: Oyunu baslat / Tekrar oyna

## Teknik Detaylar

- Dil: Saf JavaScript (ES6+)
- Render: HTML5 Canvas API
- Fizik: Ozel yercekimi ve carpisma sistemi
- Animasyon: requestAnimationFrame dongusu
- Kurulum gerektirmez - direk tarayicide ac!

## Nasil Calistirilir

1. index.html dosyasini indir
2. Tarayicida ac
3. Enter'a bas ve oyna!

## Gelistirici

Van dogumlu bir yazilim gelistirici tarafindan gelistirildi.
Python, Flask, yapay zeka ve JavaScript ile proje gelistiren
MIT hedefli portfolyo projesidir.
